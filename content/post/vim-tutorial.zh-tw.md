## 爲什麼要學 Vim

在服務器上改配置、沒有圖形界面的環境下，你只能用一個終端編輯器。Vim 是幾乎所有 Linux 發行版預裝的，學會了在任何機器上都能幹活。

它的設計哲學：**大多數時間你是在看/改代碼，不是在敲新代碼**。所以默認是"normal 模式"，鍵盤變成了一堆快捷操作。

<!--more-->

## 三種核心模式

```
        按 i / a / o          按 Esc
編輯 ←──────────── 正常 ────────────→ 命令
      ────────────▶      ←────────────
      按 Esc                 按 : / /
```

| 模式 | 作用 | 進入方式 |
|------|------|----------|
| **正常模式** | 瀏覽、刪除、複製、移動 | 按 `Esc` |
| **插入模式** | 正常打字輸入 | `i`(光標前) `a`(光標後) `o`(下一行) |
| **命令模式** | 保存、退出、搜索、替換 | `:` 或 `/` |

核心原則：**手指不離開鍵盤，持續停留在主鍵區**。

## 移動：丟掉方向鍵

```
h ← 左    j ↓ 下    k ↑ 上    l → 右

w       跳到下一個單詞開頭
b       跳到上一個單詞開頭
e       跳到單詞末尾
0       跳到行首
^       跳到行首第一個非空字符
$       跳到行尾
gg      跳到文件開頭
G       跳到文件末尾
:42     跳到第 42 行
```

爲什麼不用方向鍵？因爲方向鍵離主鍵區太遠，每次抬手浪費時間。

## 編輯操作

```vim
" 刪除
x         刪除當前字符
dw        刪除一個單詞
dd        刪除整行
d$        刪除到行尾

" 複製粘貼
yy        複製整行
yw        複製一個單詞
p         粘貼到光標後
P         粘貼到光標前

" 撤銷與重做
u         撤銷
Ctrl+r    重做

" 其他
.         重複上一次操作（非常有用的快捷操作）
>>        當前行右縮進
<<        當前行左縮進
```

## 搜索與替換

```vim
" 搜索：按 / 輸入關鍵詞，回車
/search_word     " n 下一個匹配，N 上一個匹配

" 替換全文
:%s/old/new/g    " 替換全文所有匹配

" 替換並逐個確認
:%s/old/new/gc   " y 替換，n 跳過
```

## 多文件與分屏

```vim
:split file2.py       " 水平分屏打開另一個文件
:vsplit file2.py      " 垂直分屏
Ctrl+w hjkl            " 在分屏間移動光標
Ctrl+w =               " 分屏大小均分
:bd                    " 關閉當前 buffer
```

## 實用配置

在 `~/.vimrc` 中加入：

```vim
syntax on              " 語法高亮
set number             " 顯示行號
set relativenumber     " 顯示相對行號（方便跳轉）
set expandtab          " 將 Tab 轉換爲空格
set tabstop=4          " Tab 顯示寬度
set shiftwidth=4       " 縮進寬度
set autoindent         " 自動縮進
set hlsearch           " 高亮搜索結果
set incsearch          " 增量搜索
set ignorecase         " 忽略大小寫
set smartcase          " 有大寫時區分大小寫
set mouse=a            " 支持鼠標
```

## 最少必要命令

一天學會 Vim 不太現實，但記住這 10 個操作就能幹活了：

| 操作 | 命令 |
|------|------|
| 開始編輯 | `i` |
| 退出編輯 | `Esc` |
| 保存 | `:w` |
| 退出 | `:q` |
| 不保存強制退出 | `:q!` |
| 保存並退出 | `:wq` 或 `ZZ` |
| 刪除一行 | `dd` |
| 複製一行 | `yy` |
| 粘貼 | `p` |
| 撤銷 | `u` |

## 學習路線

1. 先用 `vimtutor` 命令（Vim 自帶教程）過一遍基礎操作
2. 安裝一個終端內的練習工具（如 VimBeater）
3. 日常改配置、寫筆記時強制用 Vim，堅持兩週
4. 逐步加插件（NERDtree、fzf、coc.nvim）

別一上來就堆插件，先把**移動和編輯**的基本功練熟。
