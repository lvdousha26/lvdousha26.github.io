## なぜVimを学ぶのか

サーバー上の設定ファイルを編集するなど、GUIがない環境ではターミナルエディタを使うしかありません。VimはほぼすべてのLinuxディストリビューションにプリインストールされており、一度覚えればどんなマシンでも作業できます。

Vimの設計思想：ほとんどの時間はコードを読んだり修正したりするのであって、新しいコードを打ち込んでいるわけではありません。そのため、デフォルトは「ノーマルモード」になっており、キーボードがショートカット操作の集まりになっています。

## 3つの基本モード

```
        i / a / o を押す          Esc を押す
編集 ←──────────── 通常 ────────────→ コマンド
      ────────────▶      ←────────────
       Esc を押す              : / / を押す
```

| モード | 機能 | 切り替え方 |
|------|------|----------|
| **ノーマルモード** | 閲覧、削除、コピー、移動 | `Esc` を押す |
| **インサートモード** | 通常の文字入力 | `i`(カーソル前) `a`(カーソル後) `o`(次の行) |
| **コマンドモード** | 保存、終了、検索、置換 | `:` または `/` |

基本原則：**指をキーボードから離さず、常にホームポジションに留まる**。

## 移動：方向キーを卒業しよう

```
h ← 左    j ↓ 下    k ↑ 上    l → 右

w       次の単語の先頭にジャンプ
b       前の単語の先頭にジャンプ
e       単語の末尾にジャンプ
0       行頭にジャンプ
^       行頭の最初の非空白文字にジャンプ
$       行末にジャンプ
gg      ファイルの先頭にジャンプ
G       ファイルの末尾にジャンプ
:42     42行目にジャンプ
```

なぜ方向キーを使わないのか？方向キーはホームポジションから遠く、そのたびに手を動かすのは時間の無駄だからです。

## 編集操作

```vim
" 删除
x         删除当前字符
dw        删除一个单词
dd        删除整行
d$        删除到行尾

" 复制粘贴
yy        复制整行
yw        复制一个单词
p         粘贴到光标后
P         粘贴到光标前

" 撤销与重做
u         撤销
Ctrl+r    重做

" 其他
.         重复上一次操作（非常有用的快捷操作）
>>        当前行右缩进
<<        当前行左缩进
```

## 検索と置換

```vim
" 搜索：按 / 输入关键词，回车
/search_word     " n 下一个匹配，N 上一个匹配

" 替换全文
:%s/old/new/g    " 替换全文所有匹配

" 替换并逐个确认
:%s/old/new/gc   " y 替换，n 跳过
```

## 複数ファイルと画面分割

```vim
:split file2.py       " 水平分屏打开另一个文件
:vsplit file2.py      " 垂直分屏
Ctrl+w hjkl            " 在分屏间移动光标
Ctrl+w =               " 分屏大小均分
:bd                    " 关闭当前 buffer
```

## 便利な設定

`~/.vimrc` に追加：

```vim
syntax on              " 语法高亮
set number             " 显示行号
set relativenumber     " 显示相对行号（方便跳转）
set expandtab          " 将 Tab 转换为空格
set tabstop=4          " Tab 显示宽度
set shiftwidth=4       " 缩进宽度
set autoindent         " 自动缩进
set hlsearch           " 高亮搜索结果
set incsearch          " 增量搜索
set ignorecase         " 忽略大小写
set smartcase          " 有大写时区分大小写
set mouse=a            " 支持鼠标
```

## 最低限必要なコマンド

1日でVimを覚えるのは現実的ではありませんが、以下の10個の操作を覚えれば作業ができるようになります：

| 操作 | コマンド |
|------|------|
| 編集を開始 | `i` |
| 編集を終了 | `Esc` |
| 保存 | `:w` |
| 終了 | `:q` |
| 保存せず強制終了 | `:q!` |
| 保存して終了 | `:wq` または `ZZ` |
| 一行削除 | `dd` |
| 一行コピー | `yy` |
| 貼り付け | `p` |
| 元に戻す | `u` |

## 学習ロードマップ

1. まず `vimtutor` コマンド（Vim標準搭載のチュートリアル）で基本操作を一通り学ぶ
2. ターミナル内で使える練習ツール（VimBeaterなど）をインストール
3. 日常の設定変更やメモ書きでは強制的にVimを使い、2週間続ける
4. 徐々にプラグインを追加（NERDtree、fzf、coc.nvim）

最初からプラグインを詰め込みすぎないで、まずは**移動と編集**の基本をしっかり練習しましょう。
