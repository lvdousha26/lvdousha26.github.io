[Hexo](https://hexo.io/) へようこそ！これが最初の投稿です。詳細は [ドキュメント](https://hexo.io/docs/) をご確認ください。Hexoの使用中に問題が発生した場合は、[トラブルシューティング](https://hexo.io/docs/troubleshooting.html) で解決策を探すか、[GitHub](https://github.com/hexojs/hexo/issues) で質問できます。

## クイックスタート

### 新しい投稿を作成する

``` bash
$ hexo new "My New Post"
```

詳細：[書き方](https://hexo.io/docs/writing.html)

### サーバーを起動する

``` bash
$ hexo server
```

詳細：[サーバー](https://hexo.io/docs/server.html)

### 静的ファイルを生成する

``` bash
$ hexo generate
```

詳細：[生成](https://hexo.io/docs/generating.html)

### リモートサイトにデプロイする

``` bash
$ hexo deploy
```

詳細：[デプロイ](https://hexo.io/docs/one-command-deployment.html)
